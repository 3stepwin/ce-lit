# UX Specification: CELIT v2.0 "ALL OUT +"

## 1. Screen: Ascend (Initiation)
- **Primary Action**: "Begin Ascension" (Glow-stepped button).
- **Secondary Actions**: Role selection tiles, Face upload/selfie.
- **Micro-interactions**: Pulse on button, subtle background drift.

## 2. Screen: Generating (Liminal Wait)
- **Visuals**: Glitch text, status progress bar (10% steps).
- **Tone**: "Compliance Verification", "Archive Retrieval".

## 3. Screen: Viral Result (Artifact Delivery)
- **Top Bar**: Aesthetic Preset Tag (e.g., ANALOG ROT // VHS).
- **Main View**: High-intensity looping trailer.
- **Overlay**: Proof Artifact (Screenshot text) prominent on screen.
- **Feed**: "You are already part of this" implication banner.
- **Trinity Extensions**: Horizontal scroll of Outtakes (Locked/Gated preview).
- **Share Logic**: Caption Pack selector + Publish button.
- **Secret Reveal**: "Deleted Line" fuel (tap to copy).

## 4. Aesthetic Presets
1. **Prestige Clean**: Minimalist, serif fonts, neutral colors.
2. **Corporate Dystopia**: System fonts, flat colors, fluorescence.
3. **Analog Rot**: Glitched fonts, scanlines, timestamps.
4. **Liminal Dream**: Soft haze, italics, blurred edges.
